PCFit X

PCFit X (or previously called "PCFit" for the pygame version) is a port of popular gaming series Wii Fit by utilising both the Wii Balance Board using the Java library "WiiRemoteJ". For the Wii Remote however, Java library "motej" is used because of it's simplicity.

This port is made with libGDX, a powerful Java game engine. This port is intended for desktop platforms, mainly Linux since the accesibility of it's Bluetooth system. However, this game could also run on Windows when supported L2CAP drivers (WIDCOMM) are installed. Android development is currently out of the plan, but might happen after the base game is finished.

As of writing this part (12th of September 2023), only the basic Wii Balance Board and the GUI recreation is implemented.

GUI replication test

Though, it will be hard and dangerous to both release the game, and release the source code because of the copyright issues with Nintendo.
