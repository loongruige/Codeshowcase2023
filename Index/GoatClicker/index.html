Goat Clicker

SEE FINISHED PRODUCT

Goat Clicker is a very simple clicker game where you can click goats, and trade them... to get more goats!

Taken from Google Play Store

Goat Clicker is a idle clicker-ish game I made with my low-end computer a year ago. The main goal of the game is get goats by clicking on the goat icon in the middle of the screen and add to your goat count as you click. Then, with the said collected goats, you can get shepherds and breeders to automatically gain goats for you.

However, this app was a failure, because of the bugs. The app wouldn't run in the background without any thread permitted by the user, so getting those automatic miners were a waste.

Another failure in the game was there wasn't any competition or difficulty in it, making the gameplay boring.

Adding to the failure list, using Kotlin and the raw Android API wasn't useful for this project, and I made that failure too, a few months ago when I tried to make "Goat Clicker 2" using Flutter.

This begs the question: If I had the chance, would I make "Goat Clicker 3"? And my answer would be, that, I don't know. Now I am an intermediate libGDX game developer and I've made much more complicated projects now. But I wouldn't, because the idea isn't worth it. There is much better games that applied this idea, like Cookie Clicker, where you would actually want to continue the game, not just press some buttons a few times and uninstall it.

Technical side

Enough literature if I have to say, let's take a look at how this game works.

As I've stated, I made the game in my old computer and I probably don't have the source code anymore, so I will explain from similar code examples.

I've used SharedPreferences to store the goat, shepherd and breeder count. The game would update every second using a thread to add to the goat count by getting the shepherd and breeder count, someting like this:

                            
val mainHandler = Handler(Looper.getMainLooper())
    
mainHandler.post(object : Runnable {
    override fun run() {
        val sharedPref = activity?.getPreferences(Context.MODE_PRIVATE) ?: return
            with (sharedPref.edit()) {
                putInt("goats", getInt("goats") + (getInt("shepherd")*2 + getInt("breeder")*3))
                apply()
        }
        mainHandler.postDelayed(this, 1000)
    }
})

Of course, as I've told, it was when I first started Kotlin, I didn't know much about proper coding.

This is the main idea of the thread loop. For the clicking part, I've just attached a onClickListener to a TextView with the goat emoji, and made it increment the preference every click.

If you have a similar idea to mine, please don't do what I did. There is better ways to implement this.
